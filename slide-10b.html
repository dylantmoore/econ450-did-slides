Levels vs. Percentage Changes

Same Baseline: No Difference

When both groups start at the same level, parallel trends in levels and percentage changes are identical.

Different Baselines: It Matters

When baselines differ, the two assumptions produce different counterfactuals and different treatment effect estimates.

In practice, economists often default to assuming percentage changes are parallel (equivalent to taking logs of the outcome variable). But the right choice depends on context.

Lecture Notes: When Does This Matter?

This subtlety only arises when the treatment and control groups have different starting levels. When they start at the same place (as in the step-by-step DID example), the level change and percentage change coincide exactly.

But in most real-world applications, the groups do start at different levels. In that case, you are making a choice—often implicitly—about which version of parallel trends to assume.

Log transformation: When researchers take the log of the outcome variable before running their regression, they are effectively assuming parallel trends in percentage terms. This is very common in applied work and is often the default. But there is nothing automatically correct about it—you need a substantive reason.

The key point: there is nothing neutral here. Both assumptions are assumptions, and they give different answers. You need context-specific intuition about which one is more plausible.
