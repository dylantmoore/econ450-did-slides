Enter Difference-in-Differences

Key insight: Use a second difference (across time) to remove time-invariant selection bias

$$\text{DID} = \bigl(\color{#009E73}{Y_{\text{treat,post}}} - \color{#009E73}{Y_{\text{treat,pre}}}\bigr) \;-\; \bigl(\color{#D55E00}{Y_{\text{ctrl,post}}} - \color{#D55E00}{Y_{\text{ctrl,pre}}}\bigr)$$

First difference: removes time-invariant characteristics of each group

Second difference: removes common time trends affecting both groups

Critical assumption: Parallel trends — in the absence of treatment, both groups would have followed the same trajectory.

Lecture Notes: Why DID Is So Important

Difference-in-differences is arguably the workhorse of modern empirical economics, particularly for public policy questions. The reason is simple: we usually cannot randomly assign policies.

You cannot go to Congress and say: "Please randomly assign tax credits to some states but not others so we can learn what happens." Politicians would not think this is a great idea.

Instead, different states and jurisdictions adopt different policies at different times. DID exploits this natural variation by comparing places that adopted a policy to places that did not, before and after adoption.

The key assumption—parallel trends—says that in the absence of treatment, both groups would have changed at the same rate. This is not about having the same level of the outcome (they can start from very different places). It is about having the same trajectory.
